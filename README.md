Created desk app that is connected to web server through websocket connection.
There is web to make a reservation for users.
After user fill all necessery info the reservation is gonna be send to web server.
If no client is online web server is gonna save the reservation.
After client connected to web server is gonna send all reservation that are pending.
The desk app is gonna save received reservation except the one he already saved.
The client is able to approve or deny the reservation.
The desicion is gonna saved into json file that acts as database.
